Exercices JavaScript - jQuery

Structure du projet

Le projet est constitué des dossiers suivants :

  1. apercus qui contient des copies d'écran des résultats attendus.
  2. exercices contient les fichiers HTML de tous les exercices
  3. files contient des fichiers JSON et XML qui seront utilisés dans certains exercices.
  4. images Contient des images de l'interface.
  5. interface contient les sous dossiers css et js et html. Ce dernier contient des fragments HTML du menu et le contenu de cette page.
  6. reponses est le dossier dans lequel vous allez trvailler. Il contiendra tous les scripts JS relatifs aux exercices.

Travail à faire :

  1. Ouvrez le projet dans un éditeur de code.
  2. Lancez un serveur web de développement à la rcine de l'aplication. (Dossier contenant index.html)
  3. Pour chaque exercice, comlétez le fichier reponse/script-n.js où n correspond au numéro de l'exercice.
  4. Tous les scripts JS doivent être commentés correctement.
  5. Il est conseillé de faire vos tests sur la console devtools de Google Chrome.

Ce qu'il faut rendre :

Créer un dossier nommé par votre nom_prénom. Collez dedans les dossiers reponses et files. Zippez le dossier et envoyez-le à l'adresse email correspondant à votre classe.
