Exercice 8 XML - XSL - JSON

Enoncé

Nous voulons afficher les noms des départements français dans une liste select. Quand un département est sélectionné, on affiche toutes les villes de ce département dans une autres liste.
Pour cela, nous allons d'abords préparer les données à la fois dans un fichier XML fr.xml et dans un autre fichier JSON fr.json. Les données seront extraites du fuchier modial.xml.

  1. Créer une feuille de style XSL fr.xsl qui permet de générer le fichier fr.xml. La structure du fichier est la suivante :
    • <fr> (la racine)
      • <dep> *(plusieurs éléments département ayant un attribut nom)
        • <ville> * (plusieurs éléments ville)
  2. Créer une feuille de style XSL fr-json.xsl qui permet générer le fichier fr.json. Le fichier contiendra un tableau d'objets dep. L'objet dep a deux attributs :
    • nom (nom du département)
    • villes (tableau de toutes les villes du département)

Résultat

nom de la ville

Réponse
