Exercice 6 EVENTS

Enoncé

keypdown et keyup sont utilisé pour récupérer le code de la touche clavier. Par contre keypress permet de récupérer le code du caractère.

  1. Attacher un événement keypress à la zone de texte #f1 et analyser l'objet Event dans la console.
  2. L'attribut charCode de l'objet Event permet de récupérer le code du caractère. Faire en sorte que la div#out recopie le contenu de de #f1
  3. Appliquer un filtre au texte saisie de sorte que ne s'affichent que les chiffres.
  4. Faire la meme chose dans #f2 à la différence que la recopie ne se fasse que pour des lots de trois caractères
