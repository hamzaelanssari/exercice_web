Exercice 3 Manipulation & Ajax

Enoncé

La balise div#ce est vide. Nous voulons récuperer le fichier JSON users.json et l'afficher avec le Highlighting Javascript. Pour cela :

  1. Créer une balise <pre> et l'ajouter dans la div
  2. Ajouter la classe CSS «language-javascript» à la balise <pre>
  3. Créer une balide <code> et l'ajouter dans la balise <pre>
  4. Faire une requete AJAX pour récupérer le ficher user.json sous format texte
  5. Ajouter son contenu dans la balise <pre>
  6. Appeler la métode «Prism.highlightAll()»
